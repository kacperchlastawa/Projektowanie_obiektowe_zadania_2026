# Zadanie 1 - Sortowanie Bąbelkowe (Pascal + Docker)

Projekt realizuje algorytm sortowania bąbelkowego w języku Pascal wraz z testami jednostkowymi i konteneryzacją.

## Funkcjonalności
* Generowanie listy losowych liczb z określeniem zakresu i ilości.
* Implementacja algorytmu Bubble Sort.
* 5 testów jednostkowych sprawdzających poprawność generowania i sortowania danych.
* Pełna konteneryzacja środowiska (Docker).

## Wymagania
* Zainstalowany **Docker**.

## Instrukcja uruchomienia

### Przez skrypt Bash
